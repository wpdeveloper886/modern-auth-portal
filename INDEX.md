## 🚀 Installation

1. **Upload**: Copy `modern-auth-portal` to `/wp-content/plugins/`
2. **Activate**: Go to Plugins > Modern Auth Portal > Activate
3. **Configure**: Go to Auth Portal > Settings
4. **Create Page**: Add `[modern_auth_login]` to a WordPress page
5. **Done!**: Users can now login
